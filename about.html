About the Website

This website was created using modern web technologies to provide an immersive and interactive experience of Diwali. Here are some key features:

  • Responsive design using Tailwind CSS for a seamless experience across all devices
  • Interactive animations powered by GSAP (GreenSock Animation Platform) to bring the festival to life
  • Carefully curated content that balances mythological accuracy with contemporary relevance
  • Accessibility considerations to ensure the content is available to all users

I'm constantly working on improving this site and adding new features. If you have any suggestions or would like to collaborate on similar projects, feel free to reach out to me!
